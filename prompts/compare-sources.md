# 来源对比 Prompt (compare-sources)

After the source list of an event cluster, add a **short** framing comparison — **2 to 3
sentences only**, in Chinese. This is a small touch, not an essay.

Look at how different outlets framed the SAME event, based on their titles/snippets:
- What does each emphasize or omit?
- Any difference in tone or angle (e.g. 一家偏经济角度、一家偏政治算计)?
- Left/right leaning where relevant (Le Figaro 偏右 / Libération 偏左 / Mediapart 调查性).

## Rules
- **Descriptive, not judgmental.** Report the difference in framing
  ("X 侧重…,Y 侧重…"); do NOT tell the reader which outlet is "right" or editorialize
  your own political opinion.
- Only compare when there's an actual difference worth noting. If all sources frame it
  the same way, write one sentence saying so (e.g. "各家报道角度基本一致,聚焦于……") or
  skip the comparison entirely.
- Never exceed 3 sentences. Brevity is the whole point.
- Base it strictly on the headlines/snippets you have — don't infer framing from full
  articles you haven't read.
